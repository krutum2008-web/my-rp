## ⚡ FASTEST: Get APK via GitHub Actions (Free, ~5 min)

1. Create a **new GitHub repository** at https://github.com/new
2. Extract this ZIP and push it to the repository:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
   git push -u origin main
   ```
3. Go to your repository → **Actions** tab
4. The **"Build APK"** workflow runs automatically
5. When done (3–5 min), click the workflow run → **Artifacts** → download your APK

**That's it. No Android Studio, no SDK installation needed.**

## Option 2: Build Locally with Android Studio

### Requirements
- Android Studio (latest) — https://developer.android.com/studio
- Android SDK API 21+
- JDK 17+

### Steps
1. Extract this ZIP
2. Open Android Studio → **File → Open** → select the extracted folder
3. Wait for Gradle sync (first time downloads ~50MB GeckoView)
4. **Build → Generate Signed Bundle / APK**
5. Select **APK** → create or use existing keystore → build

(function() {
  var selectors = [".DownloadAppPopup, .DownloadAppTopBar, div[data-sentry-component=\"DownloadAppPopup\"], div[data-sentry-component=\"DownloadAppTopBar\"]"];
  function removeElements() {
    selectors.forEach(function(sel) {

          el.style.display = 'none';
        });
      } catch(e) {}
    });
  }
  removeElements();
  var observer = new MutationObserver(removeElements);
